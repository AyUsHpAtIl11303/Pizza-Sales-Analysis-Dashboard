# Pizza-Sales-Analysis-Dashboard
This project focuses on analyzing pizza sales data using Power BI to generate actionable business insights. The dashboard helps stakeholders understand sales performance, customer preferences, and operational efficiency by visualizing key metrics such as revenue, order trends, and product performance.

## 📂 Dataset Source
The dataset used in this project is sourced from Maven Analytics and is publicly available for learning purposes:

🔗<a href="https://www.mavenanalytics.io/data-playground/pizza-sales">Pizza Sales Data</a>

Alternative source:
🔗<a href="https://www.kaggle.com/datasets/nextmillionaire/pizza-sales-dataset">Alternate Source</a>

## 📊 Key Performance Indicators (KPIs)

💰 Total Revenue
📦 Total Orders
🍕 Total Pizzas Sold
📈 Average Order Value
📊 Average Pizzas per Order

## 🛠️ Tools & Technologies

Power BI – Dashboard creation & visualization
DAX (Data Analysis Expressions) – Calculations & KPIs
Power Query – Data cleaning & transformation

## 🔍 Key Insights

Identified best-selling pizza categories contributing maximum revenue
Found peak ordering hours and busiest days
Observed customer preferences for specific sizes and categories
Highlighted underperforming products

## 🚀 How to Use

Download the .pbix file from this repository
Open it using Power BI Desktop
Explore the interactive dashboard using filters and visuals
